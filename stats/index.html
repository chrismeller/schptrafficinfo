Traffic Incidents in South Carolina

During our one big winter storm in February, 2014 I discovered that the South Carolina Highway Patrol provides a website that lists all the current traffic incidents reported in the state.

One Important Note: The SCHP data does not define any sort of identifier for an "incident", so I've done the best I can to infer which records represent a distinct event. I've found that the Location reported for an incident is frequently updated after an officer arrives on the scene, so we are only able to use Incident Type, Date, Time, and County to represent a unique event. That means that if there are two wrecks reported in Greenville County at the exact same time (rush hour?) we'll have no way of knowing and would under-count the number of incidents. This also applies if the officer updates the Incident Type, which would cause us to over-count the event. As a result none of these numbers should be used independently, we can only compare the relative change over time.
